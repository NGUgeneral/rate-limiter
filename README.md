# rate-limiter

<br />
Very basic, but robust implementation of Distributed Rate Limiter.<br />
<br />
To start the project locally:<br />
<code>docker-compose up build</code><br />
<br />
To develop the project<br />
Navigate to project directory and pen 2 terminals<br />
<code>docker-compose up redis -d</ code><br />
<code>uvicorn main:app --reload</ code><br />
